# Workflows for Crypto RAG Development

## Available Workflows
Type "/" in Cascade to trigger:

### /generate_agent
Creates agent implementation with specified functionality
Example: "Generate Technical Analysis Agent with RSI/MACD calculation"

### /create_endpoint
Builds FastAPI endpoint with validation
Example: "Create /query endpoint with market analysis functionality"

### /debug_graph
Diagnoses Graphiti schema or query issues
Example: "Debug why price nodes aren't linking to prediction nodes"

### /add_datasource
Implements new data source integration
Example: "Add Coinbase Pro websocket feed to CCXT ingestion"

### /test_component
Generates unit tests for specific component
Example: "Write tests for PredictionAgent confidence scoring"
